# Code-Alpha_Credit_Scoring_System
Project Overview

Credit score prediction is an important task in financial risk assessment. This project uses Machine Learning techniques to analyze customer financial data and predict credit score categories with high accuracy.

The complete workflow includes:

Data cleaning and preprocessing

Exploratory Data Analysis (EDA)

Statistical analysis

Feature engineering

Model training and comparison

Model evaluation

Saving the best performing model

🛠️ Technologies Used

Python

Jupyter Notebook

Pandas

NumPy

Scikit-learn

Matplotlib

Seaborn

🔍 Project Workflow
1️⃣ Data Preparation

Handled missing values

Removed inconsistencies

Encoded categorical variables

Scaled numerical features

2️⃣ Exploratory Data Analysis (EDA)

Distribution analysis

Correlation heatmaps

Feature relationship visualization

3️⃣ Statistical Analysis

Summary statistics

Feature correlation analysis

Outlier detection

4️⃣ Feature Engineering

Created new meaningful features

Selected important predictors

Reduced irrelevant/noisy variables

5️⃣ Model Training

Multiple classification models were trained and compared, including:

Logistic Regression

Random Forest

Decision Tree

Other Scikit-learn classifiers

6️⃣ Model Evaluation

Models were evaluated using:

Accuracy

Precision

Recall

F1-score

Confusion Matrix

7️⃣ Saving the Best Model

The best-performing model was saved for future inference and deployment.

📈 Results

✅ Final Model Accuracy: 91.3%
